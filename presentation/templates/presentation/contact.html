Site de présentation du réseau TAROT réalisé avec Django

À propos de ce projet

Ce site web a été développé dans le cadre d'un entretien technique pour présenter le réseau des télescopes TAROT. Il démontre les compétences en développement web avec Django et l'architecture MVC.

🛠️ Technologies Utilisées

  • Django (Python)
  • HTML5 / CSS3
  • Architecture MVC
  • Design responsive
  • Navigation fluide

✨ Fonctionnalités

  • Navigation entre pages
  • Templates modulaires
  • Design moderne
  • Interface responsive
  • Code structuré
